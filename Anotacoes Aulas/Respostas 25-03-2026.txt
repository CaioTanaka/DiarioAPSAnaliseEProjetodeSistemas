Respostas Ana:
Para a realização do cadastro de um livro, é necessário ter certas informações sobre o livro, como o título, ISBN, edição entre outras. Quais informações do livro são necessárias para realizar o tombo desse livro quando ele chega na biblioteca.? Além disso, é gerado algum código de barras, QR Code para colar no livro?
R: 
Título, autor, ISBN, editora, edição, ano e número de tombo gerado pelo sistema. O número de tombo é o identificador interno de cada exemplar físico.
Sim, geramos etiqueta com código de barras pra colar na lombada. QR Code nunca usamos aqui, sempre foi código de barras mesmo.

Atualmente, como o processo de reserva é feito? Quais as informações necessárias do livro para realizar a reserva manualmente no balcão da biblioteca?
R:
Hoje é tudo no caderno mesmo. Anotamos nome do aluno, matrícula, título do livro e data da reserva. A reserva vale 2 dias úteis, se não vier buscar, cancela.

Livro de referência não entra em reserva, só consulta local
